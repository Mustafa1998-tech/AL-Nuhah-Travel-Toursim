# AL-Nuhah Travel & Tourism

موقع تعريفي لشركة **النهي للسياحة والسفر** يقدم خدمات حجز الطيران والفنادق والبكجات السياحية، مع تصميم حديث، متجاوب، ومجهز لمحركات البحث (SEO).

## محتويات المشروع

- `index.html` – الصفحة الرئيسية للموقع.
- `css/style.css` – التنسيقات الأساسية والتصميم المتجاوب (Responsive).
- `css/animations.css` – الحركات (Animations) والتأثيرات البصرية.
- `js/main.js` – التفاعلات (القائمة، السكрол، AOS، الفورم، السلايدر...).
- `images/` – الشعار والصور المستخدمة في الموقع.
- `robots.txt` – ملف توجيه عناكب محركات البحث.
- `sitemap.xml` – خريطة الموقع لتحسين الأرشفة في محركات البحث.
- `.htaccess` – إعدادات الكاش والضغط والأمان (للخوادم التي تدعم Apache).
- `.gitignore` – استثناء الملفات غير الضرورية من Git.

## المزايا الرئيسية

- تصميم متجاوب يدعم **الموبايل والتابلت والكمبيوتر**.
- تأثيرات **Animate On Scroll (AOS)** وحركات CSS مخصصة.
- تحسينات **SEO**:
  - Meta Tags كاملة (العنوان، الوصف، الكلمات المفتاحية...).
  - Open Graph & Twitter Cards لمشاركة أفضل على شبكات التواصل.
  - Structured Data من نوع `TravelAgency` وفق Schema.org.
  - ملفات `robots.txt` و `sitemap.xml` جاهزة.
- قائمة موبايل جانبية مع زر hamburger.
- زر رجوع للأعلى، و Parallax في منطقة الهيرو.

## طريقة التشغيل محلياً

1. حمّل أو استنسخ (clone) المشروع.
2. افتح الملف `index.html` مباشرة في المتصفح، أو استخدم خادم بسيط مثل:
   - VS Code Live Server
   - أو أي HTTP Server بسيط.

لا توجد حاجة إلى بناء (Build) أو أدوات Node – المشروع **Static HTML/CSS/JS**.

## النشر على Vercel

1. اربط هذا المجلد بمستودع Git (GitHub مثلاً).
2. من لوحة تحكم Vercel:
   - اختر **New Project** ثم اختر الـ Repository.
   - اختر Framework: **Other** (Static Site).
   - تأكد أن `index.html` في جذر المشروع.
3. اضغط **Deploy**.

بعد النشر، يمكنك:
- ربط الموقع بدومين مخصص مثل `alnuhah-travel.com`.
- تحديث الروابط داخل `index.html`, `robots.txt`, `sitemap.xml` لتطابق الدومين النهائي.

## ملاحظات
